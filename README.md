# 📊 Engineering Placement & Internship Analysis

### 🔹 نبذة عن المشروع (Overview)
هذا المشروع يقدم تحليلاً شاملاً لبيانات التوظيف والتدريب الصيفي لطلاب الهندسة. تم الربط بين عدة متغيرات لاستخراج رؤى (Insights) تساعد في فهم متطلبات سوق العمل الأكاديمية والمهنية.

### 🔹 الأدوات المستخدمة (Tech Stack)
* **Excel:** تنظيف البيانات وإعدادها.
* **MySQL:** بناء الاستعلامات وتحليل قاعدة البيانات.
* **Tableau:** تصميم لوحة بيانات تفاعلية (Interactive Dashboard).

### 🔹 النتائج الرئيسية (Key Insights)
* **توزيع الطلاب:** تحليل كثافة الطلاب في كل قسم هندسي.
* **الرواتب:** مقارنة متوسط الرواتب السنوية (CTC) بين الأقسام المختلفة.
* **التدريبات:** خريطة حرارية (Heat Map) توضح المجالات المفضلة للتدريب لكل تخصص.
* **الأداء الأكاديمي:** دراسة العلاقة بين المعدل التراكمي (GPA) وقيمة العروض الوظيفية.
